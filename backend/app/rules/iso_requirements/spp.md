# SPP — Generator Interconnection Requirements

## 1. Process overview & timeline

- Annual **DISIS cluster windows**; all requests in a window are studied together.
- Flow: application window → validation/deficiency cure → **DISIS Phase 1** (~180 days) → Decision Point 1 → **DISIS Phase 2 (restudy)** → Decision Point 2 → **Facilities Study** → GIA.
- Typical window-to-GIA duration ~ 2–3 years depending on restudy needs.
- Interim/provisional service (Provisional GIA) available with study caveats.

## 2. Application / entry requirements

- Application in the cluster window with:
  - Completed GI application form and technical data
  - Study deposit and readiness demonstrations (Order 2023 style)
  - **Site control** evidence (see §4)
  - POI, requested service (Network Resource Interconnection Service vs Energy Resource Interconnection Service), requested COD
- Deficiency review with a cure period; uncured applications rejected from the cluster.

## 3. Deposits, fees & milestones

- **Study deposit** scaled by size (indicative post-Order-2023 schedule: $100k–$400k by MW tranche, plus per-MW readiness security as requests advance).
- Escalating **commercial readiness deposits** at each decision point; portions become non-refundable on withdrawal after study results.
- Network upgrade financial security posted at GIA per Attachment V terms.

## 4. Site control requirements

- **100% site control for the generating facility at application** (Order 2023 compliance) — lease/option/deed covering footprint appropriate to technology, term through COD; deposit-in-lieu only for demonstrated regulatory barriers (e.g., federal lands).
- Gen-tie control or routing plan by facilities study.
- Site control affidavit with legal descriptions; acreage-vs-MW sanity checks applied in validation.

## 5. Technical data & modeling requirements

- **PSS/E** power flow and dynamics per SPP modeling requirements (MOD standards, MMWG-compatible).
- IBRs: approved second-generation renewable dynamic models or accepted vendor user models; **PSCAD** on request for weak-grid areas (SPP has extensive low-SCR regions — expect it for wind/solar/BESS in the west).
- Reactive capability ±0.95 PF at POI; PFR per Order 842; ride-through documentation.
- Short circuit data; SSR/SSO screening data near series compensation.
- One-line diagram, equipment ratings, metering and protection detail.

## 6. Required documents checklist

1. GI application (cluster window submission)
2. Study deposit + readiness security wire confirmations
3. Site control evidence (100% facility footprint) or regulatory-barrier deposit justification
4. Signatory/entity documentation
5. PSS/E raw + dyr models with documentation
6. PSCAD model (if requested / low-SCR area)
7. Reactive capability curves
8. One-line diagram
9. Site plan / boundary map (KMZ or plat)
10. Equipment data sheets (inverter, GSU, BESS)
11. NRIS/ERIS election and COD

## 7. Withdrawal & penalties

- Withdrawal at decision points: unexpended study deposit refunded; readiness deposits forfeited on an escalating schedule (higher forfeiture after Phase 2 results).
- Withdrawals triggering material restudy can carry restudy cost responsibility.
- Failure to cure deficiencies or post security → removed from cluster.

## 8. Common deficiency triggers

- Site control below 100% footprint or term short of COD.
- Model data failing SPP validation (bus/branch mismatches, unsupported models).
- Missing PSCAD data in low-short-circuit-strength areas.
- MW chain inconsistencies across application, one-line, and equipment data.
- Readiness security not matching the tranche schedule.
