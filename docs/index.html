Workflow

From drawing to filing gate

  1. 01 Register project

    Set ISO/RTO, MW, and POI substation.

  2. 02 Upload SLD revision

    Attach CAD-exported PDF and label the revision.

  3. 03 Run readiness audit

    Queue review; results return as structured findings.

  4. 04 Triage & release

    Close blockers, export report, file to the ISO portal.
