# PJM — Generator Interconnection Requirements

## 1. Process overview & timeline

- Applications accepted in **Cycle windows**; all valid applications in a window form a cluster Cycle.
- Each Cycle: **Phase 1 system impact study (~120 days)** → Decision Point I → **Phase 2 (~120 days)** → Decision Point II → **Phase 3 (~120 days)** → Decision Point III → GIA execution (Facilities Study folded into phases).
- End-to-end from window close to GIA is roughly **2 years** when the Cycle runs on schedule.
- **Surplus Interconnection Service** and **uprate** paths exist for existing points; a **Reliability Resource Initiative**-style expedited entry has been used for shovel-ready capacity (window-specific).

## 2. Application / entry requirements

- Application submitted in Queue Point during the Cycle window with:
  - Completed application form and one-line diagram
  - **Site control for the generating facility (100%)** — mandatory, no deposit alternative for the plant footprint
  - Study deposit and readiness deposit (see §3)
  - Requested service type: **Capacity Interconnection Rights (CIRs)** or Energy-only
- MW requested must be supported by equipment data; hybrid/co-located configurations declared at application.

## 3. Deposits, fees & milestones

- **Study deposit** scaled by project size (indicative: $75k ≤ 20 MW, $150k for 20–80 MW, $300k for 80–300 MW, $400k > 300 MW — verify current schedule).
- **Readiness deposits**: Readiness Deposit 1 at application and Readiness Deposit 2 at Decision Point II, each scaled to network upgrade cost exposure ($4M cap patterns apply); portions become non-refundable as the project advances through Decision Points.
- Decision Points I/II/III are the only exit ramps with partial refunds; between them deposits are at risk.

## 4. Site control requirements

- **100% site control for the generation site at application**, maintained through the process; documented by deed, lease, or option with term through expected COD.
- Indicative acreage tests used in validation (PJM Manual guidance): ~ 5 acres/MW solar, ~ 1 acre/2 MW storage containers, wind by turbine layout.
- Gen-tie / interconnection facilities: site control or routing evidence by later Decision Points.
- Site control affidavits are audited; misrepresentation grounds for rejection.

## 5. Technical data & modeling requirements

- **PSS/E** power flow raw data and dynamics; PJM requires models compatible with its base cases (MMWG format).
- IBRs: approved library dynamic models (REGC/REEC/REPC) or accepted user models; **PSCAD** data for weak-grid screens on request.
- Reactive capability ±0.95 PF at POI (Order 827); PFR per Order 842; ride-through per PRC-024 / IEEE 2800.
- Short circuit contribution data, inverter fault current characteristics.
- One-line diagram with POI, metering, demarcation, relaying, transformer data.
- Fuel/technology-specific exhibits (BESS energy duration and charging behavior).

## 6. Required documents checklist

1. Queue Point application (executed)
2. Study deposit + Readiness Deposit 1 wire confirmations
3. Site control evidence for full plant footprint (100%)
4. One-line diagram (breaker-level preferred)
5. Site plan / layout with parcel boundaries
6. PSS/E raw + dyr model data
7. Reactive capability curves
8. Equipment data sheets (inverters, GSU, BESS)
9. Corporate/signatory documentation
10. CIR vs energy-only election

## 7. Withdrawal & penalties

- Withdrawal at Decision Points: refund of the unexpended study deposit; escalating portions of readiness deposits forfeited (Decision Point II/III forfeit more than DP I).
- Withdrawal between decision points: readiness deposits substantially at risk; cost allocation for restudy impacts may apply.
- Failure to provide data or execute the GIA on schedule → deemed withdrawn.

## 8. Common deficiency triggers

- Site control acreage below technology footprint or expiring before COD.
- One-line MW inconsistent with application MW / CIR request.
- Dynamic models not in PJM-accepted library form.
- Missing BESS charging declarations for hybrid projects.
- Deposit wires not matching the applying legal entity.
